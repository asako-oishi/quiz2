Quiz - Responsive Design & Media Queries

  1. True: Your base styles should target 320px and should reside inside a media query. True
  2. False: You must re-define all styles, including those that aren't changing from the base styles, inside a media query. False
  3. What meta tag do we need in conjunction with media queries? Meta tag identify Design Screen size.
  4. What does this meta tag (the answer to # 3) do? Meta tag tells the media device to interpret the viewpoint/ browser width as the physical width of the mobile device in use.
  5. What's better to use in media queries - max/min-width or max/min-device-width? Why? Max/min-width. max/min-device-width means the actual reslution of the device and max width means the viewpoint area resolution. Width is more versatile when it comes to creating responsive webpages, though device-width is useful when you wish to specifically target mobile devices.
  6. We've been using the "screen" media type in our media queries. What's another media type we could specify? print
  7. You have 2 general options to deal with browsers that don't natively support media queries (*ahem* IE 8)? What are they? JQuery Media Queris plugin, and The Man in Blues
  8. True or false: You should structure your HTML based on importance of content, not on the designs or media queries. True
